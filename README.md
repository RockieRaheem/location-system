# Country Administrative Registration & Location Management System
This archive contains a starter implementation and all deliverables provided by the assistant:
- Import script to seed Firestore from `ug-locations`
- Firebase Cloud Functions (Express) API skeleton
- React Native (Expo) app skeleton with a LocationSelector component
- Firestore security rules
- PRD (Product Requirements Document)
- Notes, run instructions, and sample configuration

**Important**
- Replace placeholders (service account JSON, Firebase project id, API keys) before use.
- This package assumes you will install dependencies and configure Firebase/Expo locally.

Files inside:
- scripts/import_ug_to_firestore.ts
- functions/src/index.ts
- mobile-app/ (expo app skeleton)
- firestore.rules
- prd/PRD.md
- docs/NOTES.md
